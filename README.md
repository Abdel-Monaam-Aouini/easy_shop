# EASY SHOP (NodeJS ExpressJS MongoDB ReactJS)

# server

```sh
    cd server/
```

Dockerize mongodb :

```sh
    docker-compose up --build -d mongodb_easy_shop
```
Install Dependencies:

```sh
    yarn
```
OR
```sh
    npm install
```

Run Server:

```sh
    yarn start
```
OR
```sh
    npm start
```
